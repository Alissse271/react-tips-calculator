## About The Project

This project was created to calculate the amount of the bill along with the tips.
To calculate the tips, you need to enter the total amount in the field `Enter bill`, the number of customers in the field `Enter persons`, and also select the tip percentage you want to leave.
By clicking on the button `Ohhhoooo üçª`, you will receive the amount that each client must pay.

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Styled components](https://styled-components.com/)
- [React-select](https://react-select.com/home)

<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Alissse271/react-tips-calculator.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm start
   ```
